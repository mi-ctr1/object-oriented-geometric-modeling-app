Object-Oriented Geometric Modeling Application

An object-oriented application that models geometric shapes and calculates their area and volume using structured class-based design.

This project demonstrates how object-oriented programming principles can be used to represent geometric objects and perform mathematical computations.


Overview

Geometric modeling is a common application of object-oriented programming. By representing shapes as objects, programs can organize geometric properties and calculations in a structured and reusable way.

This application models geometric shapes and calculates values such as:
- area
- volume

Each shape is represented by a class that encapsulates the logic required to compute its geometric properties.


Features
- Object-oriented design
- Geometric shape modeling
- Area and volume calculations
- Encapsulation of shape properties
- Reusable class-based structure


Example Shapes Modeled
The application includes geometric objects such as:
- Rectangle
- Sphere
- Cylinder
- Cube
